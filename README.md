# Car Price Prediction with Machine Learning

## About
Predicting car selling prices using Machine Learning as part of CodeAlpha Data Science Internship June 2026.

## Libraries Used
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
- Model: Random Forest Regressor
- Accuracy: 95.83%
- MAE: 0.65 lakhs

## Features Used
- Present Price
- Driven KMs
- Fuel Type
- Transmission
- Car Age
- Owner
